# 🎓 Student Data Management System (MySQL)

## 📌 Project Overview
This project is a database system built using MySQL to manage student information efficiently. It includes database schema design, sample data, and SQL queries for data retrieval and analysis.

---

## 🎯 Objectives
- Store student details in structured format  
- Perform CRUD operations (Create, Read, Update, Delete)  
- Execute queries for data analysis  

---

## 🗄 Database Structure
The database includes tables such as:
- Students  
- Courses  
- Enrollments  

---

## ⚙️ Features
- Add, update, and delete student records  
- Retrieve student data using SQL queries  
- Manage relationships between tables  
- Perform filtering and sorting operations  

---

## 🛠 Tools & Technologies
- MySQL  
- SQL  

---

## 📂 Project Files
- `database_schema.sql` → Database structure  
- `sample_data.sql` → Sample records  
- `queries.sql` → SQL queries  

---

## ▶️ How to Run

1. Open MySQL Workbench  
2. Run `database_schema.sql` to create database  
3. Run `sample_data.sql` to insert data  
4. Execute `queries.sql` to test queries  

---

## 🔍 Sample Queries
- Retrieve all students  
- Find students by course  
- Calculate total enrollments  

---

## 💡 Use Case
This system can be used by schools or colleges to manage student records efficiently.
